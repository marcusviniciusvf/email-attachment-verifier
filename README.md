📧 Email Attachment Verifier  

Ferramenta em **Python** para análise de anexos e links suspeitos em e-mails.  
O objetivo do projeto é auxiliar na **detecção de possíveis ameaças** contidas em mensagens, automatizando a verificação de arquivos e URLs maliciosas.  

--- PROJETO EM PROGRESSO ---

## 🚀 Funcionalidades (ROADMAP)
- [x] Conexão com servidor IMAP para leitura de e-mails  
- [ ] Extração de anexos de mensagens  
- [ ] Análise de links suspeitos no corpo da mensagem  
- [ ] Integração com Virustotal API para análise de links 
- [ ] Integração com Virustotal API para análise de arquivos em anexo
- [ ] Criação de relatórios em PDF para resultado das análises
---

## 🛠️ Tecnologias Utilizadas
- **Python 3.9+**
- Bibliotecas padrão:
  - `imaplib` (conexão IMAP com servidor de e-mails)
  - `email` (processamento de mensagens)
  - `os` (manipulação de arquivos e diretórios)
  - `json` (armazenamento de resultados)
  - `datetime` (controle de datas e horários)

> ⚠️ Nenhuma dependência externa é necessária para a versão atual.
